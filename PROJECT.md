# Telescopius Plugin for TRMNL

This is a project for a plugin for the TRMNL e-Ink display, to display information from Telescopius, the Astronomy planning tool.

## Useful Resources

Telescopius API url: https://api.telescopius.com/v2.0/

Telescopius Swagger API Docs: https://api.telescopius.com/

TRMNL API Documentation: https://docs.trmnl.com/go/llms.txt

TRMNL Framework (Plugin UI System) Documentation: https://context7.com/websites/usetrmnl_framework

TRMNL local dev server: https://github.com/usetrmnl/trmnlp
